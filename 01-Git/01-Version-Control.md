# Version Control

Status : 🟡 Overview Completed

## Objective

Understand why Version Control exists and why every software company uses it.

## Overview

Version Control is a system that records every change made to files over time. It allows developers to track history, restore previous versions, and collaborate safely with multiple team members.

## Key Concepts

- Track changes
- File history
- Team collaboration
- Restore old versions
- Audit changes

## Real World

Every software company uses Version Control before deploying applications.
