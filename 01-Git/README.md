# Git

Git is a Distributed Version Control System (DVCS) used to track changes in source code, collaborate with multiple developers, and maintain the complete history of a project.

---

# Learning Objectives

By completing this section, you will be able to:

- Understand Version Control
- Install Git
- Configure Git
- Create a Repository
- Work with Working Directory
- Understand Staging Area
- Create Commits
- Work with Branches
- Merge Branches
- Resolve Merge Conflicts
- Connect Local Repository with GitHub
- Push and Pull Code
- Work with SSH Authentication
- Follow Production Best Practices
