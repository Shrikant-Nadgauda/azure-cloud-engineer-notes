### Interview Answer (30 Seconds)

Git is a Distributed Version Control System (DVCS) used to track changes in source code, collaborate with teams, and maintain project history.

---

### Detailed Explanation

Git stores every change made to files in a repository. It allows multiple developers to work on the same project without overwriting each other's work. Using features like branches, commits, and merges, Git helps manage software development efficiently.

---

### Real World Example

Imagine five Cloud Engineers working on the same Azure project. Instead of sharing ZIP files, everyone works on separate Git branches and later merges their changes into the main branch.
