## Repository Name

```

```

## Branch Naming Convention

```

```

## Project Tasks

- Repository Create करें.
- README.md तैयार करें.
- सभी Files Create करें.
- कम से कम 10 Meaningful Commits करें.
- ऊपर दिए गए सभी Feature Branches Create करें.
- प्रत्येक Branch में अलग-अलग Changes करें.
- सभी Changes Commit करें.
- सभी Feature Branches को Main Branch में Merge करें.
- कम से कम एक Branch पर Rebase करें.
- एक Commit Revert करें.
- एक Commit Reset करें.
- एक Commit Cherry-Pick करें.
- कम से कम एक Merge Conflict Create करें और Resolve करें.
- Repository को GitHub पर Push करें.
- Repository को दूसरी Location पर Clone करें.
- Clone Repository में Changes करें.
- Pull करें.
- Fetch करें.
- कम से कम 3 Tags Create करें.

## Commit Rules

- Minimum 20 Commits करें.
- सभी Commit Messages Meaningful होने चाहिए.
- Conventional Commit Naming Follow करें.

Examples

```

feat:

fix:

refactor:

chore:
```

## Branch Workflow

- सभी Development Feature Branch में करें.
- Direct Main Branch पर Development न करें.
- सभी Feature Branches को Main में Merge करें.
- कम से कम एक Branch पर Rebase करें.
- Merge के बाद Branch Delete करें.
