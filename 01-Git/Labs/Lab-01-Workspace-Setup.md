# Lab 01 - First Repository

## 🎯 Objective

इस Lab में आप Git Environment तैयार करेंगे और Project Workspace Create करना सीखेंगे।

---

# 📋 Prerequisites

- Git Installed होना चाहिए।
- Git Configuration (user.name और user.email) पहले से Configure होनी चाहिए।
- Git Bash या PowerShell उपलब्ध होना चाहिए।

---

# 📝 Task 1 - Project Workspace Create करें

नीचे दिया गया Folder Structure Create करें।

```
D:\
└── Audix-Azure-Project
    └── Lab-01
```

### Example

```powershell
mkdir D:\Audix-Azure-Project\Lab-01
```

---

# 📝 Task 2 - Project Folder Open करें

Git Bash या PowerShell से Folder Open करें।

### Example

```bash
cd /d/Audix-Azure-Project/Lab-01
```

---

# 📝 Task 3 - Git Installation Verify करें

Git Installed है या नहीं Verify करें।

### Example

```bash
git --version
```

---

# 📝 Task 4 - Git Configuration Verify करें

Git User Configuration Check करें।

### Example

```bash
git config --global user.name

git config --global user.email
```

---

# 📝 Task 5 - Current Directory Verify करें

Current Working Directory Check करें।

### Example

```bash
pwd
```

---

# 📝 Task 6 - Folder Contents Verify करें

Current Folder के अंदर मौजूद Files और Folders देखें।

### Example

```bash
ls
```

---

# 📝 Task 7 - Practice Assignment

अब नीचे दिए गए पाँच Project Workspaces स्वयं Create करें।

```
Audix-Network-Team

Audix-Terraform

Audix-Azure

Audix-DevOps

Audix-Production
```

हर Folder के लिए नीचे दिए गए Tasks पूरे करें।

- Create Folder
- Open Folder
- Verify Current Directory
- Verify Git Version
- Verify Git Configuration
- Check Folder Contents
