# GitHub

GitHub is a cloud-based platform built on top of Git that enables developers to host repositories, collaborate with teams, manage projects, automate workflows, and build professional portfolios.

---

# Learning Objectives

After completing this section, you will be able to:

- Create and manage GitHub repositories
- Build a professional GitHub profile
- Understand Public vs Private repositories
- Write professional README files
- Learn Markdown documentation
- Work with Issues and Discussions
- Create Pull Requests
- Understand Forks
- Create Releases and Tags
- Configure Branch Protection
- Learn GitHub Actions
- Build an open-source portfolio
