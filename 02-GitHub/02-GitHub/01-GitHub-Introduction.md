# GitHub Introduction

Status : 🟡 Overview Completed

## Objective

Understand what GitHub is and why it is used in modern software development.

## Overview

GitHub is a cloud platform that hosts Git repositories and enables collaboration between developers. It provides version control, project management, code reviews, automation, and documentation in one place.

## Topics Covered

- What is GitHub?
- Git vs GitHub
- Why companies use GitHub
- GitHub Architecture
- Collaboration Workflow

## Example

A DevOps engineer pushes Terraform code to GitHub. GitHub triggers an Azure DevOps Pipeline that validates and deploys the infrastructure automatically.
